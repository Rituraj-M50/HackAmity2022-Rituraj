Scope of this application

The application works with the speech or text user inputs in the English language only.

Decimals and negative numbers are not allowed. The numbers in the input text/speech can be of maximum 3 digits each.

The input expression or voice command consists of a single mathematical operator.

This application will also perform the following operations - LCM and GCD.
